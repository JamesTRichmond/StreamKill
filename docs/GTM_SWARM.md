# StreamKill GTM Delivery Swarm

## Thesis

StreamKill wins by collapsing the path from ad click to leak ledger.

The product is not a technical install. The product is a guided kill room: trust gate, scan, ledger, approval queue, guarded cancellation, proof receipt.

## Funnel

Ad or referral
→ Landing page
→ Trust gate
→ Scan approval
→ Leak ledger
→ Paid kill pass or concierge offer
→ Approved cancellation queue
→ Kill-room execution
→ Savings receipt
→ Referral prompt

## Acquisition Agents

### Bait Weaver

Job: Create ad hooks and landing-page angles.

Primary message:

"Find what is quietly billing you."

Proof message:

"StreamKill already recovered about $250/year from forgotten subscriptions in a live run."

Channels:

- Facebook ads
- TikTok/Reels clips
- Personal finance creators
- Reddit/communities where permitted
- Newsletter sponsorships
- Credit union or budgeting-app partnerships

### Trust Converter

Job: Turn fear into consent.

Assets:

- Security promise
- Privacy architecture
- Consent screen
- Plain-English data map
- Delete-my-scan promise

Core line:

"We scan for subscription evidence. You approve every cancellation."

### Onboarding Router

Job: Pick the fastest safe path.

Demo path:

- Controlled/local scan
- Redacted demo ledger fallback
- Guarded browser automation replay

Beta path:

- Invite-only guided runs
- Local-first scan where possible
- Manual fallback for unknown cancellation flows

Production path:

- Verified OAuth or local-first app
- Encrypted token handling if needed
- User-controlled deletion
- Formal privacy and security review

## Core Product Swarm

### Receipt Miner

Reads subscription-related evidence from receipts.

### Catalog Resolver

Matches evidence to known services and cancellation routes.

### Ledger Ranker

Ranks leaks by annual bleed, confidence, and safety.

### Kill Planner

Builds exact cancellation steps and safety labels.

### Consent Sentinel

Requires user approval before scan and before each cancellation.

### Browser Muscle

Executes approved cancellation paths in the user's authorized session.

### Audit Scribe

Produces proof receipt, status, and redacted logs.

## Monetization

### Free Leak Scan

User gets the ledger for free.

### Kill Pass

One-time paid execution offer.

Target price:

- $19 for up to 5 cancellation attempts

### Concierge Kill Room

High-touch beta offer.

Target price:

- $99 for guided cancellation session

### Guard Mode

Recurring monitoring and quarterly rescans.

Target price:

- $49/year

## Buyer

Initial buyer:

- Consumer who suspects subscription waste
- Non-technical user
- Feels pain from recurring charges
- Wants help, not another dashboard

Expansion buyer:

- Credit unions
- Budgeting apps
- Financial wellness platforms
- Employers
- Consumer finance creators

## Unit Economics Logic

A user who sees $250/year in waste can rationally pay $19-$99 to recover it.

The ledger is the conversion event.

The cancellation proof is the retention and referral event.

## Demo Proof

The July 3 demo should prove:

- Ad-to-ledger in under 3 minutes
- Annualized waste is visible
- User approves each kill
- Browser automation can execute or replay a guarded cancellation path
- High-risk services are blocked

## What To Cut

Cut:

- Technical install as the consumer front door
- Fully autonomous cancellation
- Raw inbox storage
- Production OAuth claims before verification
- Dashboard features that do not help the demo

Keep:

- Trust gate
- Leak ledger
- Kill approval queue
- Guarded browser execution
- Savings receipt
